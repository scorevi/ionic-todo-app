### Todo Object Fields

```typescript

  id?: string;                              // Auto-generated by Firestore
```

## ğŸ”„ CRUD Operations

### Create (C)
- Form includes all fields: title, description, priority, category, due date
- Title is required; other fields have sensible defaults
- Priority defaults to "medium"
- Category defaults to "General"
- Due date is optional
- Auto-generates createdAt and updatedAt timestamps

### Read (R)
- Fetches all todos from Firestore
- Displays active todos on main tab
- Shows completed todos on separate tab
- Real-time updates when data changes
- Handles missing fields gracefully with defaults

### Update (U)
- Edit modal pre-populates all current values
- Can modify title, description, priority, category, and due date
- Toggle completion status via checkbox
- Updates updatedAt timestamp automatically

### Delete (D)
- Swipe left on todo item to reveal delete button
- Permanently removes todo from Firestore
- No confirmation dialog (can be added if needed)
